# programu_uzturesana
Programmēšanas tehniķa kurss
 Nosaukums - SYSTEM HEALTH CHECKER
APRAKSTS:
Bash skripts, kas pārbauda sistēmas veselību.
Pārbauda diska vietu, operatīvo atmiņu un CPU slodzi.

LIETOŠANA:
1. chmod +x system_health.sh
2. ./system_health.sh              # Pamata pārbaude
3. ./system_health.sh quick        # Ātra pārbaude
4. ./system_health.sh full         # Pilna pārbaude
5. ./system_health.sh full report  # Pārbaude ar atskaiti

KO PĀRBAUDA:
1. Diska vietu (/)
2. Operatīvo atmiņu (RAM)
3. CPU slodzi
4. Sistēmas informāciju (full režīmā)
